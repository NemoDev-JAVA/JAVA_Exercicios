# 📚 Dominando Java - 100+ Exercícios Resolvidos  

Este repositório contém minha resolução dos exercícios propostos no livro **_Dominando Java: 100+ Exercícios Resolvidos e Comentados para acelerar seu aprendizado_**.  
O objetivo deste projeto é consolidar fundamentos da linguagem Java e demonstrar minha evolução prática como desenvolvedor.  

---

## 🎯 Objetivos do Projeto  
- Reforçar conceitos fundamentais de Java 
- Praticar Lógica de programação e Estruturas de dados
- Construir um portfólio público de aprendizado  
- Aplicar boas práticas de codificação e organização de projetos  

---

## 🛠️ Tecnologias Utilizadas  
- Java 11 (OpenJDK 11.0.28)  
- VS Code como IDE principal  
- Terminal Linux (Pop!_OS) para compilação e execução
